# Sportovní šermířský klub Rajhrad — statický web (GitHub Pages)

Tento balík obsahuje jednoduchý statický web připravený pro rychlé nasazení na **GitHub Pages**.

## Obsah
- `index.html` — hlavní stránka (single-page web s kotvami)
- `styles.css` — stylování
- `sitemap.xml` — jednoduchá sitemap
- `images/` — složka pro fotografie (prázdná, přidej svoje obrázky)
- `README.md` — tento soubor

## Jak nasadit na GitHub Pages
1. Vytvoř si účet na https://github.com pokud ho ještě nemáš.
2. Vytvoř nový repozitář, např. `sportovni-serm` (nebo `tveuzivatelskejmeno.github.io` pokud chceš mít root adresu).
3. Nahraj obsah tohoto balíku (včetně složky `images`) do repozitáře.
4. Pokud používáš běžný repozitář:
   - V GitHub → Settings → Pages vyber branch `main` (nebo `master`) a složku `/ (root)`.
   - Po chvilce bude stránka dostupná na `https://<tveuzivatelskejmeno>.github.io/<repozitar>/`.
5. Pokud jsi vytvořil repozitář pojmenovaný `tveuzivatelskejmeno.github.io`, stránka bude na `https://tveuzivatelskejmeno.github.io/`.

## Přidání fotografií
- Nahraj fotky do složky `images/` v repozitáři.
- V souboru `index.html` přidej do sekce Fotogalerie tagy `<img src="images/nazev.jpg" alt="...">`.

## SEO a indexace
- Stránka obsahuje základní meta tagy (title, description, robots).
- Pro lepší indexaci přidej stránku do Google Search Console a do Sklik/Seznam Webmaster nástrojů.

## Dotazy
Kdybys chtěl, mohu:
- Přidat tvoje obrázky do galerie za tebe.
- Pomoci s připojením vlastní domény.
- Upravit barevné schéma nebo texty.
